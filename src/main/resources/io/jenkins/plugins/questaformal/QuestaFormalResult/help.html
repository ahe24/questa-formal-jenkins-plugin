This plugin parses and displays Questa Formal verification results, including:

  • Lint Report: Displays design quality metrics, errors, warnings, and information items.
  • CDC Report: Shows Clock Domain Crossing analysis results.

Configuration

  • Lint Report Path: Path to the Questa Formal Lint report file (default: Lint_Results/lint.rpt)
  • CDC Report Path: Path to the CDC analysis report file (default: CDC_result/cdc.rpt)

All paths should be relative to the workspace directory. Absolute paths are not recommended.
